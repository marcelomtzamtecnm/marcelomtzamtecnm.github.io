Responsive web design o tambien conocido como diseño web adaptable, es una tecnica de diseño y desarrollo web que mediante el uso de estructuras flexibles (contenedores flexibles, imagenes y video flexibles) y junto con media queries especificados en CSS, logran adaptarel sitio web al entorno del dispositivo en el que se encuentre. con esta tecnica se consigue que el contewnido del diseño de tu sitio se vea bien y pueda ser bastante legible para los usuarios sobre el dispositivo con el que este interactue
